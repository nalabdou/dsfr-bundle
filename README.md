Symfony bundle for the **French Government Design System** (DSFR v1.14).

> 🚧 Work in progress — v1.0.0 expected Q3 2026

## Requirements

- PHP ^8.3
- Symfony ^6.4 || ^7.4
- symfony/ux-twig-component ^2.13

## Installation

```bash
composer require nalabdou/dsfr-bundle
```
